# Weather Application

## Description
This weather application is a simple and intuitive tool for checking current weather conditions for different locations, It fetches data from a reliable weather API and displays it in a user-friendly interface.

## Features
- Current Weather conditions(temperature, humidity, windspeed etc.)
- Search functionality for finding weather infromation by city name
- Responsive design for use on both desktop and mobile devices

## Technologies Used
- HTML5
- CSS3
- JavaScript(ES6+)
- Weather API (OpenWeather API)
